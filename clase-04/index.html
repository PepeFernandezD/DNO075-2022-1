Aprendiendo HTML

José Pedro Fernández

Para este ejercicio incrustamos/vinculamos archivos SVG en un HTML para aprender las distintas maneras de visualizar la infografía digitalmente. En el caso particular traté sobre la energía, visualizando un gráfico no figurativo del uso de energías en USA, un gráfico figurativo con una ilustración energética y un gráfico de dos capas con respecto al uso de energía renovable.

Cabe recalcar lo largo del codigo del SVG incrustado como tercera imagen en el HTML, donde existe un espacio de mejora, pero debido a las modificaciones que realizé en Illustator (máscara de recorte, fuentes, elementos, transparencia y demases) el profesor me confirmó que no habría problema.

¡Gráfico no figurativo incluido en la página como si se tratara de un objeto!

¡Gráfico figurativo incluido en la página como si se tratara de una imagen!

¡Gráficos de dos capas incrustarse como código SVG directamente en el documento HTML!
